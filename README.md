# QueryPulse Enterprise Platform

Enterprise-grade PostgreSQL monitoring and analytics platform built using Spring Boot, PostgreSQL, JWT Authentication, and modern backend architecture principles.

---

# Features

## Authentication & Security
- User Registration
- User Login
- JWT Authentication
- Password Encryption using BCrypt
- Protected APIs
- Stateless Authentication
- Global Exception Handling

---

## Backend Architecture
- Spring Boot 3.5
- Layered Architecture
- REST APIs
- DTO Pattern
- Service Layer
- Repository Layer
- Centralized Exception Handling
- Modular Package Structure

---

## Database
- PostgreSQL
- JPA / Hibernate
- UUID-based Entity IDs
- Query Optimization Ready
- Enterprise Schema Design

---
